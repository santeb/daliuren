# 大六壬在线占卜

基于中国传统六壬术的在线起课工具，支持完整的九宗门算法。

## 功能

- 🌟 **四柱八字** — 年、月、日、时干支自动推算
- 🌙 **月将换将** — 按中气精确换将（十二将）
- 🔄 **天盘排列** — 月将加时法
- 📖 **四课推导** — 干上神、支上神递推
- 🎯 **三传计算** — 九宗门全覆盖（贼克、比用、昴星、涉害、遥克、伏吟、反吟、别责、八专）
- ⚔️ **天将排盘** — 贵人起法完整，昼贵/夜贵自动判断
- 🔮 **神煞系统** — 德神、禄神、驿马、桃花等
- 📊 **长生十二宫** — 阴阳干顺逆分途
- 📝 **详细解读** — 吉凶评分、五行生克、应期方位

## 使用

直接在浏览器中打开 `daliuren_full.html` 即可使用。

在线访问：https://santeb.github.io/daliuren/

## 技术

纯前端实现，单文件 HTML，无依赖。
